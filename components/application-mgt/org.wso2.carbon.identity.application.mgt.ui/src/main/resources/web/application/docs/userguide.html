Service Providers

This user guide describes the Service Providers Management related configurations available in the WSO2 Identity Server.

You can use the left hand side "Main" -> "Identity" -> "Service Providers" menu to complete the below mentioned configuration details.

Service Providers that are application that depend on WSO2 Identity Server for various functionality and are typically registered by application developers. These Service Providers can be selectively associated to a set of Identity Providers available in the system for many different use cases. E.g. Outbound Provisioning to an external Identity Provider.

Following section describes adding and modifying Service Providers in the system by an application developer. Click on Main > Identity > Service Providers > Add to add Service Providers to the system. You can also edit and delete registered Service Providers by going to Main > Identity > Service Providers > List and clicking the corresponding links against them.
