Information Cards Issuer Configuration

The Information Cards issuer component of the WSO2 Carbon enables to configure cards issuer settings

Figure 1: Cards Issuer Configuration

  • Card Name: Display name of the downloaded Information Card
  • Valid Period: Valid period of an issued card in number of days
  • Sipporting Token Types: Tokens types being supported
  • Symmetric binding used: Specifies whether to use symmetric binding or not
