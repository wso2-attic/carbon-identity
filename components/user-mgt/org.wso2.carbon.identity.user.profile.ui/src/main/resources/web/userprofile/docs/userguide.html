User Profile Management

The User Profile Management component of the WSO2 Carbon facilitates the management and control of user's details. The key functionalities of this component include:

  • Add new profiles (Note : New profiles can only be added if the Identity Server is connected to a JDBC user store)
  • Add and modify details of a user in profile

The user profile component can be invoked in two ways.

  • Administrator or a person with "Manage Security" functionality can add/edit/view profiles of users
  • If the product is configured, the user himself can edit his profile
External profiles of a JDBC user store can be viewed only
